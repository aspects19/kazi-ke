Update the color scheme in `styles.css`:

```css

    --primary-color: #0e4fc0;
    --secondary-color: #15161e;
    --accent-color: #f59e0b;
```

Adjust animation timing and effects in `styles.css`:

```css
    animation: fadeInUp 0.6s ease-out; /* Change duration here */
}
```
